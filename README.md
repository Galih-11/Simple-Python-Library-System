# 📌 Tentang Project

Project ini dibuat sebagai implementasi konsep dasar:

* Pemrograman berorientasi objek (OOP)
* Class dan inheritance
* Manajemen data sederhana
* Sistem interaksi berbasis terminal / CLI

Aplikasi memungkinkan:

* Mahasiswa meminjam dan mengembalikan buku
* Admin mengelola koleksi buku
* Sistem menampilkan data buku dan riwayat sederhana secara interaktif
