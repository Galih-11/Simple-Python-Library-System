# Simple Python Library System

A simple object-oriented library management system built with Python that allows students to borrow and return books, and administrators to manage book collections.

## ‚ú® Main Features

* **Student Management**: Student data management with NPM, faculty, and study program information.
* **Book Management**: Add, update, delete, and display book collections.
* **Borrowing System**: Students can borrow and return books efficiently.
* **Admin Controls**: Full administrative control over library inventory.
* **Interactive Menu**: User-friendly text-based console interface.

## üõ† Technologies Used

* **Python 3.x**: Core programming language.
* **Object-Oriented Programming (OOP)**: Utilizes classes and inheritance for structure.
* **Console Interface**: Simple and lightweight CLI (Command Line Interface).

## üìã Prerequisites

* Python 3.6 or higher.
* No external dependencies required (uses standard libraries).
